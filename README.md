# Agentic Design Patterns

Agentic Design Patterns: A Hands-On Guide to Building Intelligent Systems by Antonio Gulli

# Installation

To install the dependencies for this project, you will need to use the `--legacy-peer-deps` flag with npm. This is due to some dependency conflicts that cannot be resolved otherwise.

```bash
npm install --legacy-peer-deps
```
